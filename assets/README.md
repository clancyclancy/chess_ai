# Assets Directory

This directory is for chess piece images and other graphical assets.

## Future Enhancements

You can replace the text-based piece rendering with actual images:

1. Add PNG images for each piece (WP.png, WN.png, etc.)
2. Use SDL_Image library to load textures
3. Update `GUI::renderPiece()` to render textures instead of text

## Recommended Asset Sources

- Create your own pixel art pieces
- Use free chess piece sprites from OpenGameArt.org
- Generate vector graphics and export as PNG

## File Naming Convention

- WP.png - White Pawn
- WN.png - White Knight
- WB.png - White Bishop
- WR.png - White Rook
- WQ.png - White Queen
- WK.png - White King
- BP.png - Black Pawn
- BN.png - Black Knight
- BB.png - Black Bishop
- BR.png - Black Rook
- BQ.png - Black Queen
- BK.png - Black King
